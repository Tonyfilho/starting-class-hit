Complete Guide: Angular Reactive Forms

Understanding FormControl, FormGroup, Validators

Learn how Reactive Forms work in Angular: form creation, validation rules, and control access.

Reactive Forms Overview

1. What is a Reactive Form?

Reactive Forms are structured and scalable forms created in the TypeScript code using the FormControl, FormGroup, and FormBuilder APIs. They offer full control over validation, events, and dynamic behavior.

✅ Ideal for complex or dynamic forms with advanced validation logic.

2. FormControl: The Core Building Block

A FormControl represents a single input field. It tracks the value, status, and validation state.

  • value: the current value
  • valid: boolean indicating if all validators pass
  • touched / dirty / pristine: interaction states
  • errors: object with validation errors (e.g. {{"{"}}required: true {{"}"}})

Example:


import { FormControl } from {{'@angular/forms'}};

const name = new FormControl('', { nonNullable: true });

3. Validators

Angular provides built-in validators such as Validators.required, Validators.minLength(), and Validators.email.

Attach them to a control:


import { Validators } from {{'@angular/forms'}};

const email = new FormControl('', [Validators.required, Validators.email]);

❌ If a validator fails, email.errors will contain the corresponding error key.

4. FormGroup: Combining Controls

A FormGroup groups multiple controls and allows validation and value tracking for the entire form. 


import { FormGroup } from {{'@angular/forms'}};

const loginForm = new FormGroup({
  email: new FormControl('', [Validators.required, Validators.email]),
  password: new FormControl('', [Validators.required, Validators.minLength(6)]),
});

Usage: loginForm.value, loginForm.valid, loginForm.get('email')

5. Accessing and Displaying Errors

Inside templates, use the control reference to check validation states: 


<input [formControl]="loginForm.get('email')" />

<div *ngIf="loginForm.get('email')?.errors?.['required']">
  Email is required.
</div>
<div *ngIf="loginForm.get('email')?.errors?.['email']">
  Invalid email format.
</div>

Summary

  • ✅ Use FormControl for each field
  • ✅ Group them using FormGroup
  • ✅ Add built-in or custom validators
  • ✅ Check errors to display user feedback
