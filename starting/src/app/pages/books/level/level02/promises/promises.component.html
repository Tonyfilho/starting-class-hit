Exemplo de Promise no Angular

Este exemplo demonstra como trabalhar com Promises no Angular, incluindo tratamento de erros com a sintaxe async/await.


Demonstração de Promise Usando Then e Catch

Clique no botão para simular uma requisição assíncrona:

@if (this.localLoad) {
Processando...
} @if (mensagem) {
✓ {{ mensagem }}
} @if (erro) {
✗ {{ erro }}
}

Demonstração de Promise Usando Async & Await
