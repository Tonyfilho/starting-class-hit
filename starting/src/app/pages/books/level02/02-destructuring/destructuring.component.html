JavaScript: Destructuring

O que é Destructuring?

Destructuring é uma técnica do ES6 que permite extrair dados de arrays ou objetos e atribuí-los a variáveis de forma concisa.

Destructuring com Objetos

Permite extrair propriedades específicas de um objeto:

const user = {{"{"}} name: 'Ana', age: 25 {{"}"}};
const {{"{"}} name, age {{"}"}} = user;

console.log(name); // Ana
console.log(age);  // 25

Destructuring com Arrays

Permite extrair valores posicionais de um array:

const fruits = ['🍎', '🍌', '🍇'];
const [first, second] = fruits;

console.log(first);  // 🍎
console.log(second); // 🍌

Renomeando variáveis

Você pode renomear as variáveis ao desestruturar:

const user = {{"{"}} id: 1, username: 'admin' {{"}"}};
const {{"{"}} username: name {{"}"}} = user;

console.log(name); // admin

Valores padrão

É possível definir valores padrão durante a desestruturação:

const settings = {{"{"}} theme: 'dark' {{"}"}};
const {{"{"}} theme, language = 'pt-BR' {{"}"}} = settings;

console.log(theme);    // dark
console.log(language); // pt-BR

Também pode ser usado diretamente nos parâmetros de funções:

const user = {{"{"}} name: 'Lucas', age: 30 {{"}"}};

Destructuring Aninhado

Você pode extrair dados de estruturas mais complexas:

  name: 'João',
  contact: {{"{"}}
    email: {{'joao@email.com'}},

  {{"}"}}
{{"}"}};

const {{"{"}} contact: {{"{"}} email {{"}"}} {{"}"}} = person;
