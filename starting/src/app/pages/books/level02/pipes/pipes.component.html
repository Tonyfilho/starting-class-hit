Pipes

Os Pipes no Angular são utilitários que permitem formatar ou transformar dados diretamente nos templates de maneira declarativa. Eles são usados para modificar a exibição de valores sem alterar os dados originais no componente.

Principais Características dos Pipes ✔ Transformação de dados: Convertem números, datas, strings, JSON e outros formatos. ✔ Uso em templates: Aplicados com o operador | (pipe). ✔ Encadeamento: Vários pipes podem ser usados juntos (ex: {{"{{ valor | pipe1 | pipe2 }}"}}). ✔ Puros e impuros: Controlam quando a transformação é recalculada (performance).

Regex

Site de Referência: Regex-generator

Regex (ou Expressões Regulares) é uma sequência de caracteres que define um padrão de busca em textos. É uma ferramenta poderosa para validar, extrair, substituir ou manipular strings de forma eficiente.

O que Regex faz? Busca Encontra trechos de texto que seguem um padrão específico.

Validação Verifica se uma string corresponde a um formato (ex.: e-mail, NIF, senha).

Substituição Altera partes do texto com base em regras definidas.

Extração Separa dados estruturados de um bloco de texto.

1. Pipes Built-in (Integrados)

O Angular já inclui pipes prontos para casos comuns:

Preço: {{ this.price | currency:"EUR"}}

Data: {{ this.totay | date:'dd/MM/yyyy'}}

Texto Caixa Alta: {{ text | uppercase }}

2. Pipes Personalizados

O Desenvolvedor pode criar seus Pipes:

{{"este é o exemplo do nosso pipe upcaseFirstWord" | upcaseFirstWord }}
