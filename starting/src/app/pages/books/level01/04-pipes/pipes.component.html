Angular Pipes: Data Transformation in the Template.

📌 O que são Pipes?

Pipes em Angular permitem transformar visualmente os dados no template sem modificar o valor original. São muito usados para formatação de textos, números, datas, moedas, etc.

📦 Sintaxe Básica

{{ "valor | nomeDoPipe "}}

📅 Exemplos com Pipes Comuns

// date
{{ today | date:'fullDate' }} // Ex: terça-feira, 16 de julho de 2025

// uppercase / lowercase
{{ nome | uppercase }}   // 'JOÃO'
{{ nome | lowercase }}   // 'joão'

// titlecase
{{ frase | titlecase }}  // 'Olá Mundo Do Angular'

// number
{{ numero | number:'1.2-2' }} // 1.000,00

// currency
{{ preco | currency:'EUR':'symbol' }} // €12,00

// percent
{{ percentual | percent }} // 0.75 → 75%

🧠 Composição de Pipes

Você pode usar múltiplos pipes em sequência:

{{ nome | lowercase | titlecase }}

🛠️ Criando um Pipe Personalizado

Exemplo de um pipe que deixa a primeira letra de cada palavra em maiúscula:

{{"import { Pipe, PipeTransform } from '@angular/core';"}}

{{"@Pipe({ name: 'upcaseFirstWord' })"}}
export class CapitalizePipe implements PipeTransform {{"{"}}
  transform(value: string): string {{"{"}}
    return value.replace(/\b\w/g, char => char.toUpperCase());
  {{"}"}}
{{"}"}}

// Uso:
{{ 'ola mundo angular' | upcaseFirstWord }} // 'Ola Mundo Angular'

📌 Pipes vs Métodos no Componente

Evite usar métodos diretamente no template se eles forem pesados ou causarem re-renderizações. Pipes são otimizados para isso.

🧩 Pipes Puros vs Impuros

  • Puros: são executados somente se o valor de entrada mudar.
  • Impuros: são executados a cada detecção de mudança. (Use com cautela)
