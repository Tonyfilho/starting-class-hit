Rule 8: No More Than Two Instance Variables

Simplify class design by limiting state

Em Angular, componentes com muitos atributos de instância tornam-se difíceis de testar e manter. Agrupe e encapsule sempre que possível.

Exemplo no Angular

Evite:

this.name
this.email
this.phone
this.address
this.company

Prefira:

{{"this.userDetails = { name, email, phone, address, company }"}}
