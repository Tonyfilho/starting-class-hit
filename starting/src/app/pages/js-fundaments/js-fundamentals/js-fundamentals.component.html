JS/TS Fundamentals: Data Types, Variables, Operators, and Control Structures

1. Data Types (Tipos de Dados)

JavaScript e TypeScript suportam vários tipos de dados:

  • string: texto
  • number: números inteiros e decimais
  • boolean: verdadeiro ou falso
  • array: lista de elementos
  • object: estrutura com pares chave-valor
  • null e undefined

2. Variables (Variáveis)

Use let para variáveis mutáveis e const para constantes:


x = 10;        mais usado, escopo global
let y = 20;     // escopo de bloco
