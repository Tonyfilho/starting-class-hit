JS/TS Fundamentals: Data Types, Variables, Operators, and Control Structures

1. Data Types

O TypeScript oferece tipos como: string, number, boolean, array, e any:


nome: string = 'Maria';
idade: number = 30;
ativo: boolean = true;
lista: number[] = [1, 2, 3];
qualquerCoisa: any = 'texto'; // cuidado com `any`!

2. Variables

Use let para variáveis mutáveis e const para valores fixos:


 x = 10;     //  escopo Glocal
let y = 20;     // escopo de Bloco
const z = 30;   // valor constante, escopo de Bloco Obs: arrays [] e objetos {{"{}"}} com comportamento diferente

3. Operators

Aritméticos, Comparação, Lógicos e Ternário


// Aritméticos
soma = 10 + 5;
multiplicacao = 4 * 2;

// Comparação
igual = 5 == '5'; // true
estritamenteIgual = 5 === '5'; // false

// Lógicos
e = true && false;
ou = true || false;

// Ternário
idade = 18;
podeEntrar = idade >= 18 ? 'Sim' : 'Não';
