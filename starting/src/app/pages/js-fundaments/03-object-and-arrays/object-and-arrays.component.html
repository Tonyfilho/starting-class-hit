Objetos e Arrays: Estruturas, Métodos e Práticas

1. O que são Objetos e Arrays?

Objetos e arrays são estruturas de dados fundamentais no JavaScript e TypeScript:

// Objeto
const pessoa = {{"{"}}
  nome: 'Ana',
  idade: 30,
  cidade: 'Lisboa'
{{"}"}};

// Acesso às propriedades
console.log(pessoa.nome); // Ana
console.log(pessoa['idade']); // 30

// Array
const frutas = ['maçã', 'banana', 'uva'];

// Acesso aos elementos
console.log(frutas[0]); // maçã

2. Métodos Comuns de Arrays

Os métodos abaixo são amplamente utilizados para manipular arrays. A tabela a seguir resume seus comportamentos:

Método Modifica o Array Original? Retorno Descrição
push Sim Novo tamanho Adiciona elementos ao final. 
const a = [1,2];
a.push(3); // [1,2,3]
pop Sim Elemento removido Remove o último elemento. 
const a = [1,2,3];
a.pop(); // [1,2]
shift Sim Elemento removido Remove o primeiro elemento. 
const a = [1,2,3];
a.shift(); // [2,3]
unshift Sim Novo tamanho Adiciona ao início. 
const a = [2,3];
a.unshift(1); // [1,2,3]
map Não Novo array Transforma os itens. 
[1,2,3].map(x => x * 2); // [2,4,6]
filter Não Novo array Filtra os itens. 
[1,2,3].filter(x => x > 1); // [2,3]
reduce Não Valor acumulado Reduz a um único valor. 
[1,2,3].reduce((a,b) => a + b, 0); // 6
find Não Primeiro valor encontrado Retorna o primeiro que satisfaz a condição. 
[1,2,3].find(x => x > 1); // 2

3. Diferenças Entre Objetos e Arrays

Objetos e arrays têm finalidades e estruturas diferentes, embora ambos sejam tipos de referência em JavaScript:

Característica Objeto Array
Organização Pares chave-valor Lista ordenada de valores
Acesso aos dados obj.chave ou obj['chave'] array[índice]
Ordem Não garantida Ordenada
Uso típico Representar entidades com propriedades Listas ou coleções de itens
Iteração for...in, Object.keys() for, forEach, map, etc.
Boas Práticas Evitar iterar diretamente com for...in em objetos complexos, prefira Object.entries() Evitar mutar arrays com métodos que modificam o original, prefira versões imutáveis quando possível

Exemplos práticos:

// Objeto: representando uma pessoa
const pessoa = {{"{"}}
  nome: 'Carlos',
  idade: 40,
  ativo: true
{{"}"}};
console.log(pessoa.nome); // Carlos
console.log(Object.keys(pessoa)); // ['nome', 'idade', 'ativo']

// Array: lista de idades
const idades = [21, 35, 40];
console.log(idades[1]); // 35
console.log(idades.map(i => i + 1)); // [22, 36, 41]

4. Desestruturação de Objetos e Arrays

A desestruturação permite extrair dados de objetos ou arrays de forma concisa.

Desestruturação de Objetos

const pessoa = {{"{"}}
  nome: 'Carlos',
  idade: 28,
  cidade: 'Porto'
{{"}"}};

const {{"{"}} nome, idade {{"}"}} = pessoa;
console.log(nome); // Carlos
console.log(idade); // 28

Desestruturação de Arrays

const cores = ['vermelho', 'verde', 'azul'];

const [primaria, secundaria] = cores;
console.log(primaria); // vermelho
console.log(secundaria); // verde

Renomeando variáveis

const produto = {{"{"}}
  preco: 100,
  disponivel: true
{{"}"}};

const {{"{"}} preco: valor, disponivel: emEstoque {{"}"}} = produto;
console.log(valor); // 100
console.log(emEstoque); // true

Resumo Técnico

  • ✅ Objetos armazenam dados em pares chave-valor.
  • ✅ Arrays organizam coleções ordenadas de valores.
  • ✅ push, pop, shift e unshift modificam o array original.
  • ✅ Métodos como map, filter, reduce e find retornam novos arrays ou valores sem modificar o original.
  • ✅ Conhecer os efeitos colaterais dos métodos ajuda a manter o código previsível.
  • ✅ A desestruturação facilita o acesso e a clareza no uso de objetos e arrays.
