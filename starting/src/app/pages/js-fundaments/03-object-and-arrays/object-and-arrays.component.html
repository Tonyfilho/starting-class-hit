Objetos e Arrays: Estruturas, Métodos e Práticas.

1. O que são Objetos e Arrays?

Objetos e arrays são estruturas de dados fundamentais no JavaScript e TypeScript:

// Objeto
const pessoa = {{"{"}}
  nome: 'Ana',
  idade: 30,
  cidade: 'Lisboa'
{{"}"}};

// Acesso às propriedades
console.log(pessoa.nome); // Ana
console.log(pessoa['idade']); // 30

// Array
const frutas = ['maçã', 'banana', 'uva'];

// Acesso aos elementos
console.log(frutas[0]); // maçã

2. Métodos Comuns de Arrays

Os métodos abaixo são amplamente utilizados para manipular arrays. A tabela a seguir resume seus comportamentos:

Método Modifica o Array Original? Retorno Descrição
push() ✅ Sim Número (novo comprimento) Adiciona um ou mais elementos ao final do array.
pop() ✅ Sim Elemento removido Remove o último elemento do array.
shift() ✅ Sim Elemento removido Remove o primeiro elemento do array.
unshift() ✅ Sim Número (novo comprimento) Adiciona um ou mais elementos no início do array.
filter() ❌ Não Novo array Retorna um novo array contendo apenas os elementos que passam em um teste.
map() ❌ Não Novo array Cria um novo array com os resultados da função aplicada a cada item.
find() ❌ Não Elemento encontrado ou undefined Retorna o primeiro elemento que satisfaz a condição fornecida.
every() ❌ Não true ou false Verifica se todos os elementos do array satisfazem a condição.
some() ❌ Não true ou false Verifica se pelo menos um elemento satisfaz a condição.
reduce() ❌ Não Qualquer valor Aplica uma função a um acumulador e a cada elemento do array (da esquerda para a direita), reduzindo-o a um único valor.
includes() ❌ Não true ou false Verifica se um determinado valor está presente no array.

Resumo Técnico

  • ✅ Objetos armazenam dados em pares chave-valor.
  • ✅ Arrays organizam coleções ordenadas de valores.
  • ✅ push, pop, shift e unshift modificam o array original.
  • ✅ Métodos como map, filter, reduce e find retornam novos arrays ou valores sem modificar o original.
  • ✅ Conhecer os efeitos colaterais dos métodos ajuda a manter o código previsível.
